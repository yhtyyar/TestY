==============
Вкладка Labels
==============

На вкладке **Labels** доступно управление лейблами, которые используются
для разметки тест-кейсов. На вкладке представлены все лейблы,
созданные в проекте:

|image20|

При создании нового лейбла доступен выбор цвета, который в дальнейшем используется в качестве фона для созданного лейбла.
