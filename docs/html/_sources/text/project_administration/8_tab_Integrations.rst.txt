Вкладка Integrations
=============================

На вкладке **Integrations** доступно управление ссылками на внешние ресурсы, которые в дальнейшем попадают в интеграционное меню на странице :ref:`<test_plans_results>`.

Для добавления новой ссылке перейдите на вкладку **Integrations** и нажмите на кнопку "+ Create Integration". В открывшемся окне заполните поля:
- Name (имя интеграции, которое в дальнейшем будет отображаться в интеграционном меню)
- Description (описание интеграции)
- Service URL (ссылка на внешний ресурс)

При заполнении Service URL допустимо использование плейсхолдеров TESTY_PROJECT_ID и TESTY_PLAN_ID в качестве значений атрибутов GET запроса. 

Переключатель "Open in new tab" по умолчанию установлен в положение On. Переключатель отвечает за поведение при нажатии на ссылку. 
Во включенном состоянии ссылка будет открываться в соседней вкладке.
